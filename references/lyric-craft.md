# Lyric Craft

Open this when writing or revising lyrics for stronger hooks, rhyme, line length, originality, language fit, ReMi-style drafting, or singability.

## Writing Procedure

1. Define the song's one sentence promise.
2. Choose the chorus hook before filling verses.
3. Make each verse add new information rather than repeating the same feeling.
4. Keep lines short enough to sing in one breath.
5. Use concrete images instead of abstract emotional labels.
6. Revise away cliches before final output.
7. If adapting an existing draft, preserve the best hook or image and change only the weak parts.

## Section Jobs

| Section | Job |
|---------|-----|
| Verse 1 | Establish scene, speaker, conflict, or desire. |
| Pre-Chorus | Raise emotional pressure or reveal stakes. |
| Chorus | State the memorable hook in repeatable language. |
| Verse 2 | Add consequence, contrast, or a new detail. |
| Bridge | Change angle, time, key emotion, or decision. |
| Final Chorus | Return bigger, clearer, or more resolved. |

## Singability

| Problem | Fix |
|---------|-----|
| Lines read like prose | Break into 4-9 word phrases. |
| Chorus has no hook | Repeat a strong title phrase or melodic chant. |
| Verse says the same thing twice | Replace one line with a physical image or action. |
| Too many concepts | Keep one central metaphor or situation. |
| Weak rhythm | Use parallel sentence shapes and end rhymes. |

## Rhyme Options

| Type | Use |
|------|-----|
| End rhyme | Clear pop memorability. |
| Slant rhyme | More natural modern phrasing. |
| Internal rhyme | Rap, R&B, energetic verses. |
| Repeated phrase | Hooks, chants, and post-chorus sections. |

## Cliche Watchlist

Avoid default AI phrases unless the user explicitly wants them:

```text
tapestry, echoes, whispers, neon lights, shadows, dreams,
fly, free, moment, darkness, light, unfold, embark, scars,
fire inside, rise above, breaking chains, in the night
```

Replace with sensory specifics:

| Weak | Stronger direction |
|------|--------------------|
| "lost in the night" | name the street, room, weather, object, or time |
| "chasing dreams" | describe the actual risk or action |
| "heart is broken" | show the habit, silence, message, or absence |

## Language Notes

- Mandarin lyrics should favor clean rhythm, vivid nouns, and repeatable hook phrases.
- English lyrics should avoid crowded multisyllabic lines unless rap-oriented.
- Cantonese lyrics need extra attention to natural phrasing; avoid literal Mandarin-to-Cantonese translation.
- Bilingual songs should assign language by section or hook, not random line switching.
- If the user wants Suno to draft inside the app, describe the lyric brief they can put into `Write with Suno`/ReMi, but still provide a finished lyric when asked.

## Rewrite Modes

| User asks for... | Change | Keep |
|------------------|--------|------|
| Catchier chorus | Shorten hook, repeat title phrase, sharpen rhyme. | Verse story and style unless flawed. |
| Less generic | Replace abstract nouns with objects, places, actions, and stakes. | Structure and emotional intent. |
| More commercial | Add cleaner hook, simpler section jobs, stronger final chorus. | Language and core theme. |
| Same song, new verse | Match meter, perspective, and rhyme density. | Existing chorus and title. |
| Continuation | Start from the last emotional state, not the beginning. | Prior language, tempo, and motif. |

## Safety And Originality

- Do not reproduce existing song lyrics.
- If the user asks for a named artist's style, translate it into neutral traits such as tempo, instrumentation, era, vocal energy, and production texture.
- For parody or homage, keep wording and melody-independent phrasing original.
